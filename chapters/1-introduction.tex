\section{Problem Formulation}

Consider a pipe with a known length, $L$, and known friction coefficient, $\theta$, as defined by \cref{eq:theta_defined}. The pipe may contain $n$ leaks at unknown locations $z_1,\dots,z_n$, and where the shape and size of the $i$-th leak is characterised by an unknown constant $c_i$. \\

Available measurements of the pipe system are limited to steady-state measurements of volumetric discharge and piezometric head at the inlet and outlet of the pipe. By operating the system at different steady-states, multiple measurements can be obtained to construct the measurement set $\mathcal{M}$.\\

Let $\mathcal{F} : \{(z_i,c_i)\}_{i=1}^n \mapsto \mathcal{M}$ denote the forward operator that maps the leak parameters to the corresponding set of measurements. In this work, the measurement set $\mathcal{M}$ is generated by simulation, where the pipe parameters $L$ and $\theta$ are known exactly, i.e., there is no model error. The measurements are therefore assumed to be free of both model error and measurement noise.\\


The leak identification problem is then defined as solving the inverse problem
\begin{equation*}
    \mathcal{F}\ ^{-1} :  \mathcal{M} \mapsto \{(z_i,c_i)\}_{i=1}^n.
\end{equation*}



Under these assumptions, the following two subproblems of the leak identification problem are considered:

\renewcommand{\descriptionlabel}[1]{\normalfont #1}
\begin{description}[itemsep=2pt, topsep=0pt, parsep=0pt]
    \item[\textbf{Problem 1}] Given that the number of leaks, $n$, is known. Determine under which conditions the leak identification problem can be solved, and implement a method to solve the problem when conditions are met.

    \item[\textbf{Problem 2}] Given that the number of leaks, $n$, is unknown. Develop a method that determines the number of leaks in the pipe from an available measurement set $\mathcal{M}$.

\end{description}
